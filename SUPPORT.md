# Support

## How to file issues and get help

This project uses GitHub Issues to track bugs and feature requests. Please search the existing
issues before filing new issues to avoid duplicates. For new issues, file your bug or
feature request as a new Issue.

For help and questions about using this project, please check:

1. **README documentation** - Start with the main [README.md](README.md) for setup and usage instructions
2. **Microsoft Documentation** - Visit [Microsoft Entra External ID documentation](https://learn.microsoft.com/entra/external-id/)
3. **Native Auth API Reference** - See [Native Authentication API Reference](https://learn.microsoft.com/entra/external-id/customers/reference-native-authentication-api)

## Microsoft Support Policy

Support for this project is limited to the resources listed above.
